2. Cüzdan Kümeleme Motoru (Wallet Clustering)
2.1 Algoritma Detayı
Cüzdan kümeleme, aynı kişiye ait birden fazla cüzdanı graf analizi ile tespit eder. Dolandırıcının 400 sahte cüzdanı aslında tek bir kümedir.
Adım 1 — Funding Graph Oluşturma: Her holder cüzdanı için ilk SOL transferini geriye doğru izle. Kaynak cüzdan → hedef cüzdan kenarı oluştur. Bu işlem recursive olarak 3 seviye derinliğe kadar yapılır.
Adım 2 — Timing Cluster: Aynı 5 dakikalık pencerede oluşturulan cüzdanları grupla. 10+ cüzdan aynı zaman diliminde oluşturulmuşsa = script ile üretilmiş.
Adım 3 — Behavioral Similarity: Benzer miktarlarda (±%5 tolerans), benzer zaman aralıklarında işlem yapan cüzdanları eşleştir. Cosine similarity ile davranış vektörü karşılaştırması.
Adım 4 — Connected Components: networkx kütüphanesi ile grafın bağlantılı bileşenlerini bul. Her bileşen = bir küme = muhtemelen aynı kişi.

3. Wash Trading Tespit Motoru
3.1 Cycle Detection Algoritması
Amaç: A → B → C → A gibi döngüsel işlemleri tespit etmek. Bu, aynı paranın hacim şişirmek için döndürüldüğünü kanıtlar.
def detect_wash_cycles(transactions, max_depth=5):
"""Direkt graf üzerinde 2-5 adımlık döngüleri bul"""
G = build_transfer_graph(transactions)
cycles = []
for node in G.nodes():
for cycle in nx.simple_cycles(G):
if len(cycle) <= max_depth:
cycles.append({
'wallets': cycle,
'total_volume': sum_cycle_volume(cycle, transactions),
'avg_interval': avg_time_between(cycle, transactions),
'amount_variance': calc_variance(cycle, transactions)
})
return cycles

4.3 Kademeli Çıkış Tespiti (Laddered Exit)
Algoritma: Büyük holder cüzdanlarının satış geçmişini izle. 3+ satış işlemi, her biri fiyatı %10+ düşürüyor, aralarında fiyat toparlanması var = kademeli çıkış.
def detect_laddered_exit(whale_wallet, token_address, price_history):
sells = get_sells(whale_wallet, token_address)
if len(sells) < 3: return None
drops = []
for sell in sells:
price_before = get_price_at(sell['timestamp'] - 60)
price_after = get_price_at(sell['timestamp'] + 60)
drop_pct = (price_before - price_after) / price_before
if drop_pct > 0.10:  # %10+ düşüş
drops.append({'sell': sell, 'drop': drop_pct})
if len(drops) >= 3:
return LadderedExitResult(detected=True, stages=drops)
